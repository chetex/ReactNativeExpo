# Pokedex - React Native CRUD App

A simple yet polished Pokemon manager built with React Native and Expo. Features full CRUD operations, Material Design-inspired UI, and official Pokemon artwork from PokeAPI.

> Built entirely with **Claude Opus 4.6** (Anthropic's AI coding assistant).

## Features

- **Browse** — Scroll through a pre-loaded list of 15 Pokemon with official artwork
- **Search & Filter** — Real-time search by name, type, or description
- **View Details** — Tap any Pokemon to see its full artwork, type badge, and description
- **Add Pokemon** — Create new entries via a bottom-sheet form with type selector and sprite preview
- **Edit Pokemon** — Update name, description, type, or sprite of any existing Pokemon
- **Delete Pokemon** — Remove Pokemon with a Material Design confirmation dialog (rounded modal, colored accent, Pokemon sprite)

## Screenshots

The app uses a **Pokeball-red header**, **light gray background** (`#ECEFF1`), and **type-colored accents** across all cards and badges (fire = orange, water = blue, grass = green, etc.).

## Architecture

The app follows a **component-based architecture** with a single source of truth:

- **App.tsx** — Holds all state (`pokemonList`, `selectedPokemon`, modal visibility) and CRUD logic. Passes data and callbacks down to child components.
- **Components** — Pure presentational components that receive props and emit events. No local state except form inputs in `PokemonFormModal`.
- **Types** — Shared type definitions, color constants, and utility functions extracted to `src/types/pokemon.ts`.

### Data Flow

```
App (state + CRUD logic)
├── SearchBar        → onChangeText
├── PokemonDetail    → onEdit, onDelete
├── PokemonList
│   └── PokemonCard  → onPress, onEdit, onDelete
├── PokemonFormModal → onSave, onClose
└── DeleteConfirmModal → onConfirm, onCancel
```

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Expo Go app on your phone (or an emulator/simulator)

### Installation

```bash
# Clone and navigate to the project
cd 01-initial-blank-app

# Install dependencies
npm install

# Start the development server
npx expo start
```

### Running

- **iOS Simulator**: Press `i` in the terminal
- **Android Emulator**: Press `a` in the terminal
- **Physical Device**: Scan the QR code with Expo Go

## Design Decisions

- **No external UI library** — All Material Design styling is achieved with built-in React Native `StyleSheet`, shadows, border radius, and elevation. This keeps the bundle size minimal.
- **PokeAPI sprites** — Official artwork loaded from GitHub-hosted PokeAPI sprites. No local image assets needed.
- **Custom delete modal** — Replaced native `Alert.alert` with a fully styled Material Design dialog featuring the Pokemon's sprite, type-colored accent bar, rounded corners (28px radius), and proper button hierarchy.
- **Type color system** — All 15 Pokemon types have their canonical color, used consistently across badges, card borders, and detail accents.
